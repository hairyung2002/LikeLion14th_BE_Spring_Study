# 🌱 Spring Study Repository

## 스터디 목표

1. 스프링 기반 축제 사이트 개발
2. 스프링 기반 중커톤 참여
3. 스프링 중심 기술 스택 전환

---

## 스터디 진행 방식

- 주 1회 비대면 스터디 진행
- 정해진 강의를 사전에 수강 후 참여
- **스터디 전까지**
    - 강의 수강 완료
    - 실습 진행
    - Git에 학습 내용 업로드

---

## Repository 구조

```
spring-study/
 ├── README.md
 ├── docs/                # 개념 정리, 발표 자료
 ├── common/              # 공통 실습 코드
 │    ├── week1/
 │    └── week2/
 ├── personal/            # 개인 학습 공간
 │    ├── {name}/
 │    │    ├── week1/
 │    │    └── week2/
```

### 📁 사용 방법

- `common/`
    - 강의 기반 공통 실습 코드
    - 기준이 되는 코드
- `personal/`
    - 개인 실습 및 추가 학습
    - 자유롭게 수정/확장

👉 **공통 코드와 개인 코드를 반드시 분리해주세요.**

---

- 코드 한 줄 한 줄 이해
